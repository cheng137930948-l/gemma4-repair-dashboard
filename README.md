# 🛠️ 智能产线维修综合看板 (Gemma 4 参赛演示版)

这是一个为电子制造工厂量身定制的**实时维修数据可视化平台**。它将传统的 MES 维修报表与 AI 模型相结合，旨在提升 PCB 维修效率和工时管理透明度。

### 🌟 核心亮点
* **实时数据对接**：自动读取 MES 维修报表，实时生成 HTML 综合看板，告别手动统计。
* **维修逻辑辅助**：集成 Gemma 模型，针对 PCB 常见故障（如 LLC 炸管、TL431 异常等）提供维修建议。
* **多维度可视化**：
  * **工时达成率看板**：实时监控维修人员产出。
  * **故障分布统计**：自动归纳“撞件PAD起来”等高频机械性损伤。
* **离线部署优化**：支持 libs 库离线加载，适应工厂内网特殊环境。

### 🚀 技术架构
* **后端**: Python (集成 Report Server)
* **前端**: HTML5, CSS3, JavaScript (含 SVG 动态绘图)
* **AI 驱动**: Google Gemma 4 (用于故障诊断辅助)
* **自动化**: 基于 Python 的数据清洗与看板实时推送

### 📁 文件说明
* `index.html`: 项目入口，集成维修数据中心门户。
* `report_server.py`: 后端逻辑，处理 MES 数据对接。
* `material_dashboard.js`: 看板核心逻辑与 UI 交互。
* `data/demo/`: 提供演示用的维修结构参考数据。
